# Hugo 上线

把多年没有更新的 hyde 转到了 hugo 平台，过程还算顺利，md 的格式也差别不大。

# Cloudflare

选择 cloudflare 的 pages 主要有两个好处，一是方便，自动从 github 部属，不用写 action；另外是速度比较快，不会像 github 似的经常无法访问。
